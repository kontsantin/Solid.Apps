Цель проекта

C помощью простой и вовлекающей механики привлечь пользовательский траффик, который в дальнейшем будет перенаправлен на основной продукт (мобильное приложение).

PictoTapp

Цель этого Mini App - с помощью простой и вовлекающей механики привлечь пользовательский траффик, который в дальнейшем будет перенаправлен на основной продукт (мобильное приложение).

Функционал

Фотоквесты

Заводятся определенные задания (квесты), для выполнения которых пользователям необходимо сфотографировать определенный предмет.

Голосования

Другие пользователи могут голосовать за лучшую из 10 фотографий, сделанных в рамках одного фотоквеста.

Встроенная камера

Пользователи имеют возможность использовать камеру своего устройства, не выходя из Mini App.

Лидерборд

Реализован общий рейтинг и отдельный рейтинг по уровням игроков: Beginner, Amateur, Semi-pro, Pro, Guru.

2-ух уровневая реферальная система

Система рефералов для привлечения новых пользователей и мотивации существующих игроков.

Задания

Список заданий по типу «Подписаться на Telegram-канал», или «Поставить лайк на пост в Twitter».

Административная панель

С помощью встроенной панели администратора можно отслеживать статистику по пользователям.
